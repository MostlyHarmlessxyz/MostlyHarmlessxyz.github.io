离散数学

Chapter3 关系

关系的概念

$R: X\times Y \to \left {是, 否\right}$ 此处$R$就是一个二元关系/ 笛卡尔乘积的子集$2^{X \times Y}$

$若(x,y)\in R,则称x,y符合关系R，记作 xRy。若(x,y)\notin R,则称x,y符合不关系R，记作 x\cancel{R}y$

多值函数的推广: $2^{X \times Y} \leftrightarrow \left {f|f:X \rightarrow 2^Y \right}$ Proof: 构造映射，证明单射&满射

$R^{-1} = \left { (y,x)|(x,y) \in R\right} 称为R的逆。其中R\subseteq X\times Y $

$I_x = \left { (x,x) | x \in X\right } 称为恒等关系$

一些需要注意的点：

  1. 二元运算不是二元关系。前者是“3+4=7”，后者是“3和4是模n同余的吗？”
  2. 任何一个x不一定有y与之配对，任何一个y不一定有x与之配对

几种特殊的二元关系

自反关系、反自反关系

$若\forall x \in X, 都有(x,x)\in R 即 xRx 则称R是X上的自反二元关系，或称R是自反的$

$若\forall x \in X, 都有(x,x)\notin R 即x\cancel{R}x 则称R是X的反自反二元关系，或称R是反自反的$

对称关系、反对称关系

$R\subseteq X \times Y, x,y\in X: \若xRy则yRx,则称R是X上的对称二元关系；\若xRy且yRx，则x=y 则称R是X上的反对称二元关系$

注意这里，如果一个关系使得*$若xRy且yRx$*为假，则其必为反对称关系。（真值表）

如果出现一个偏的，即可说明该关系是不对称的。($\exists x\ne y, xRy$)

空集既是对称的，也是反对称的

传递关系

$设R\subseteq X\times X. 若xRy且yRz，则有xRz$

$设R\subseteq X\times X. 若R=\emptyset,则称R为空关系。其是 反自反的、对称的、反对称的、传递的二元关系$

相容关系

自反且对称的关系称之为相容关系

关系的集合运算

$设R,S\subseteq X \times Y, 则 R\cap S, R\cup S \subseteq X \times Y \ x R\cup Sy(或(x,y)\in R \cup S) \Leftrightarrow xRy 或 xSy \ xR\cap Sy \Leftrightarrow xRy且xSy\ xR|Sy \Leftrightarrow xRy 且 x\cancel{S}y$

关系的合成

$设R\sube X\times Y, S \sube Y \times Z. X \times Z的子集\left { (x,z) | \exist y \in Y ,s.t (x,y) \in R 且 (y,z) \in S\right } 称为\R与S的合成二元关系，记作R \circ S$

y就是所谓的关键人物

一些基本的性质不再赘述 结合律 分配律 穿脱法则 抽屉原理

关系合成运算的性质

$传递\Leftrightarrow R^2\sube R\对称\Leftrightarrow R^{-1} \sube R \ 自反 \Leftrightarrow I_x \sube R \ 对称且传递 \Leftrightarrow R = R\circ R^{-1} \对称且反对称 \Leftrightarrow R \sube I_x$

二元关系的闭包

传递闭包

求间接联系 最小的那个

$传递闭包R^{+} = \cap_{R\sube R’且R’是传递的} R’ \R^+ = \cup_{n=1}^{\infty}R^n \ Proof: 左侧是包含所有通过传递性推导出来的关系的最小关系，它包含了所有通过\有限次应用R连接的对，右侧是以任意步数进行连接的关系\ 便于运算的版本:|X|=n, 则 R^+ = \cup_{i=1}^{n} R^i$

证明的本质还是利用容斥原理，以及数量有限 -> 操作次数有限

$R^+(运算)保持了2^{X\times X} 上的\sube的单调性\ if R\sube S, then R^+\sube S^+\R,S\sube X\times X, \quad R^+ \cup S^+ \sube (R\cup S)^+$

自反闭包 对称闭包

不自反？把$R^{0}$并上！ 不对称？把$R^{-1}$并上！
